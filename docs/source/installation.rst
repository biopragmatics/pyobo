Installation
============

The most recent release can be installed from `PyPI <https://pypi.org/project/pyobo>`_
with uv:

.. code-block:: console

    $ uv pip install pyobo

or with pip:

.. code-block:: console

    $ python3 -m pip install pyobo

Installing from git
-------------------

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/biopragmatics/pyobo.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/biopragmatics/pyobo.git

Installing for development
--------------------------

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/biopragmatics/pyobo.git
    $ cd pyobo
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
